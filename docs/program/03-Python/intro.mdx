# 知识导航

欢迎来到教程专区 🎉 这是知识导航，开始学习～ 这里你可以找到：

### 🚀 Python 拓展学习资料：

- Python 之旅：https://funhacks.gitbooks.io/explore-python/content/ （适合0基础 入门阶段使用）
- Python 100天学习指南：https://github.com/jackfrued/Python-100-Days （适合0基础入门开始学习，流程完整）

- Python 标准库：https://docs.python.org/zh-cn/3/library/index.html

- Python 最佳实践指南：https://pythonguidecn.readthedocs.io/zh/latest/
- Python CookBook：https://python3-cookbook.readthedocs.io/zh-cn/latest/index.html （已经有Python基础后进行学习）

---

### 📦 需要学习的三方Python库列表：

- FastAPI：https://fastapi.tiangolo.com/ （WebAPI 接口层框架）
    - Starlette：https://starlette.dev/ （Web框架，提供了异步 Web 框架的基础设施，负责路由、请求处理和响应。）
    - Pydantic：https://docs.pydantic.dev/latest/ （数据验证库，Web场景下提供序列化等能力，LLM 大模型场景下稳定模型输出）

- SQLAlchemy：https://www.sqlalchemy.org/ （数据库ORM框架）
