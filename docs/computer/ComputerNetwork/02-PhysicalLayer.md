# 物理层

---

## 物理层标准定义

物理层解决如何在连接各种计算机的传输媒体上传输数据比特流，而不是指具体的传输媒体。物理层的主要任务描述为：确定传输媒体的借口的一些特性：

- 机械特性：接口形状，大小，引线数目
- 电气特性：规定电压范围（-5V到+5V）
- 功能特性：规定-5V表示0，+5V表示1
- 过程特性：也称规程特性，规定建立连接时各个相关部件的工作步骤



## 数据通信基础知识

一个数据通信系统可划分为三大部分，即**源系统**(发送方)、**传输系统**(传输网络)和**目的系统**(接收方)。
