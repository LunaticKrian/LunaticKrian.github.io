# 小红书模版

```text
看图的思维逻辑，每次看一张好图时，问自己这 5 个问题：

1. 构图：主体在画面什么位置？用了什么构图法则（三分法、引导线、对角线、框中框）？
2. 光影：光从哪来？硬光还是柔光？明暗对比大不大？
3. 色彩：主色调是什么？是互补色、类似色还是单色搭配？（你日志里已经学了这些，直接用上）
4. 情绪：这张图传达了什么感觉？是通过什么元素传达的？
5. 如果是我拍，我会怎么做？ — 这是最重要的一步

📷 构图分析
[比如：三分法构图，人物放在右侧交叉点，左侧大面积留白营造孤独感，视线从左到右自然流动]

💡 光影分析
[比如：侧逆光从左后方打来，勾出人物发丝轮廓，面部一半亮一半暗，氛围感直接拉满]

🎨 色彩分析
[比如：整体冷色调，主色蓝+青，点缀暖色光源，冷暖对比让画面很有层次感]

🎈 情绪分析
[比如：通过压暗背景表现环境的寂静气氛]

🎁 拍摄分析
[比如：我应该怎么拍才能拍出这样的感觉]

📌 本期收获
[比如：以前拍cos总习惯把人物放正中间，现在才懂留白也是一种构图语言]

图源：[出处]
```

📷 构图分析
使用了三角构图，人物的主体姿势通过手臂抱花构成三角形。
中心偏右构图 + 左侧视觉引导：人物主体位于画面中心偏右位置，怀抱的蓝色花束向左侧延伸，形成视觉重心的平衡。右侧人物占据主要画面，左侧以花枝和留白呼应，既突出人物主体，又避免了拥挤感，让视线自然从左侧花枝流向人物面部，营造出温柔又疏离的氛围。

💡 光影分析
光线从左上方打过来，使用顶光（硬光、不需要色片，白色）
侧逆光 + 局部柔光：光源从人物左后方打来，形成强烈的轮廓光，精准勾勒出白发丝、蕾丝裙摆和花束的边缘，让主体从暗背景中 “浮” 起来；面部和胸前区域有柔和补光，既保留了暗调氛围感，又清晰呈现了面部神态与服饰细节，明暗对比强烈却不生硬，高级感拉满。

🎨 色彩分析
低饱和冷色调 + 暖光点缀：主色调为黑白（服饰）+ 冰蓝色（花束），整体偏冷；侧逆光的暖金色光线打在发丝、裙摆和花束上，形成冷暖对比，既清冷又温暖，弱化了纯冷调的疏离感，让画面更有层次和故事感。

🎈 情绪分析
温柔孤寂的氛围感：暗背景压缩了环境信息，突出人物与花束的互动，低头垂眸的姿态 + 怀抱花束的动作，传递出内敛、沉静的情绪；侧逆光的朦胧感进一步弱化了尖锐感，整体像在诉说一段温柔又孤独的心事，让观者能共情到人物的细腻情绪。

🎁 拍摄分析
布光：用一盏主灯打侧逆光（人物后上方 45°），再用小柔光箱补光面部 / 胸前，避免面部过暗；背景用纯黑或深灰背景布，减少干扰。
构图：人物不要放正中间，偏右 1/3 处，让道具（花束）向留白处延伸，引导视线。
参数：用大光圈（f/1.8-f/2.8）虚化背景，ISO 控制在 100-400，快门保证人物清晰，突出轮廓光的质感。
引导：让模特低头垂眸，怀抱道具贴近身体，营造内敛的情绪感。

📷 构图分析
使用了三角构图，人物的主体侧身姿势通过手臂伸展构成三角形。
提线木偶式构图 + 视觉引导：人物占据画面中心偏下位置，上方和两侧的白色提线（应该是后期处理的）从画面顶端垂落，贯穿人物四肢，形成强烈的 “被操控” 视觉符号。提线将视线从人物延伸至画面外，既强化了木偶主题，又让画面充满张力与叙事感，打破了常规居中构图的呆板。

💡 光影分析
冷调侧光 + 低照度氛围：光源从人物侧前方打来，用冷蓝色调渲染整体氛围，明暗对比柔和却清晰，突出女仆装的蕾丝褶皱、发丝纹理和肢体轮廓。背景被压暗成深灰，进一步聚焦人物主体，营造出幽闭、压抑的剧场感，完美呼应 “提线木偶” 的无力感。

🎨 色彩分析
单色调冷蓝 + 黑白对比：整体以低饱和冷蓝色为主，覆盖人物与背景，营造清冷、忧郁的基调；服饰的黑白撞色在冷光下形成柔和对比，白色提线在暗背景中格外醒目，成为视觉焦点，既强化了主题，又让画面层次分明。

🎈 情绪分析
压抑与挣扎的矛盾感：提线、低头垂眸的姿态、冷暗环境共同传递出 “被束缚、身不由己” 的情绪，而抬起的手臂又暗含一丝挣扎与反抗，让画面充满故事性，观者能共情到角色的无助与不甘，氛围感极具穿透力。

🎁 拍摄分析
布光：用一盏带蓝色色片的柔光箱打侧光，控制光比，避免过曝；背景用纯黑 / 深灰背景布，减少环境干扰。
道具：用白色鱼线或棉线模拟提线，从天花板垂落，系在模特手腕、脚踝处，注意线条要直且分布自然。（这里如果前期没有合适的场景使用提线，需要依靠模特能力进行后期处理）
引导：让模特模仿木偶僵硬的肢体动作，低头、眼神放空，强化 “被操控” 的神态。
参数：小光圈（f/4-f/5.6）保证提线和人物都清晰，ISO 400-800 适配暗环境，快门速度保证肢体不糊。
后期：需要使用白色画笔，将提线进行重绘处理

📌 本期收获
以前拍 cos 更多的是在关注人物本身，而道具 + 光影 + 构图的叙事性才是氛围感的核心：一根简单的提线，就能让画面从 “好看” 变成 “有故事”，用视觉符号传递情绪，比单纯摆姿势更有感染力。

📷 构图分析
对角线分割（使用武器作为分割线）
中心动态构图 + 强烈纵深感：人物采用中心偏左的紧凑构图，肢体舒展且充满张力。手持的发光长剑呈对角线斜切画面，从左下延伸至右上，瞬间建立起强烈的空间纵深感与动感。周围的金色线条环绕交织，形成放射状的视觉动线，将观众的视线牢牢锁定在人物核心，极具视觉冲击力。

💡 光影分析
自发光高光 + 硬边轮廓光：拍摄打光需要从右上方使用暖色硬光打亮模特面部，清晰勾勒出人物的肢体轮廓与衣料质感；面部与身体受光均匀，高光区域明亮通透，阴影部分保留了细腻的肤色过渡。武器的金色光芒是一个发光源（武器光源如果拍摄前期没有合适的道具，就需要后期加上武器光效，暖色特效）。这种布光方式在二次元插画中表现为高光溢出的华丽感，让人物在纯黑背景中瞬间 “跳” 出来。
头部背后发丝光，可以使用一个逆光束光，局部打亮发丝。

🎨 色彩分析
高饱和金白撞色：整体色调以纯白与鎏金为主，搭配深邃的纯黑背景，形成极高的色彩对比度。金色的光带在画面中穿梭，不仅丰富了层次，更赋予画面神圣、高贵且华丽的史诗感，没有多余的杂色干扰，视觉极其干净且高级。

🎈 情绪分析
神圣与战斗的交织：冷冽的绿色眼神与微抿的嘴唇传递出一丝冷峻与决绝，而华丽圣洁的白金色服饰又暗含了 “神性” 与 “高贵”。整体情绪呈现出一种优雅的战斗力，既美丽又具有强大的压迫感，仿佛下一秒就要展开决战。

🎁 拍摄分析（仿拍建议）
这属于二次元 / 高概念商业布光，实拍很难 1:1 还原，但可借鉴思路：
布光：使用黑色吸光布背景。在武器上贴荧光条或用LED 灯带模拟自发光，面部用大柔光箱正面补光，压低环境光，侧上方使用暖色硬光打亮模特刘海，可以轻微过曝制造亮部对比。头部背后发丝光，可以使用一个逆光束光，局部打亮发丝。
后期：在修图时，单独提亮武器与装饰的高光，做出发光效果；压暗背景，使用 “外发光” 图层样式模拟插画感。
构图：大胆使用对角线构图，让道具（剑）斜穿画面，能瞬间提升画面的动感与故事性。

PS：这幅插画的拍摄手法可以和第一期积累的作品做一个很好的参考，旋转后布光方式类似。
