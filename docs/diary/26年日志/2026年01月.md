# 2026年01月

## 2026年01月02日

**合成摄影的工作流程**

1. 设计分镜动作
2. 确定背景
3. 按照背景拍摄人物素材
4. 将人物素材与背景融合

PS：将可控性低的环境安装在前面，流程顺序能保证光线与透视统一

适合新手的合成练习：平视视角、中长焦（仰视或者俯视，搜集背景的难度更大；合成广角需要扎实的透视知识）

越是具有明显方向性的光线越容易模拟，复杂技能特效光（不建议新手）
